# Contributing

Thank you for your interest in this project.

This repository documents the analytical workflow supporting the research study:

**Beyond Awareness: Understanding the Drivers of Sustainable Practice Adoption Among Built Environment Professionals**

## Reporting Issues

If you identify errors, inconsistencies, or opportunities for improvement, please open a GitHub Issue describing the problem in sufficient detail.

## Suggestions

Suggestions relating to methodology, visualization, reproducibility, or documentation are welcome.

## Scope

This repository is intended primarily as a reproducible research project and portfolio demonstration. Contributions should therefore preserve the integrity of the original analyses while improving transparency, documentation, or usability.

Thank you for your interest in improving reproducible urban sustainability research.
